📌 Project Overview

This project is a Machine Learning-based News Prediction application developed using Python and Streamlit. The application analyzes news data and predicts the category or authenticity of news articles using trained ML models.

The main goal of this project is to demonstrate how Machine Learning can be applied to text data for news classification and prediction.

🚀 Features
News prediction using Machine Learning
User-friendly Streamlit web interface
Data preprocessing and cleaning
Text feature extraction
Model training and prediction
Fast and interactive results
🛠️ Technologies Used
Python
Streamlit
Pandas
Scikit-learn
NumPy

⚙️ Installation
1. Clone the Repository
git clone https://github.com/your-username/news-prediction.git
2. Navigate to the Project Folder
cd news-prediction
3. Install Required Libraries
pip install -r requirements.txt
▶️ Run the Project
streamlit run News_prediction.py
